# **App Name**: GauRakshak

## Core Features:

- Animal Registration: Record government tag number, tag color, breed, color, health status, and animal image for each cow and buffalo.
- Movement Tracking: Track animal movement including entry date, reason for entry, exit date, and reason for exit.
- Financial Records: Manage receipts, payments, expenses, milk records, and bank records (DR/CR).
- Reporting & Analytics: Generate financial reports and enable year-end data closing.
- Data Backup: Securely back up all data.
- User Management: Admin role with multiple user roles, signup with admin approval, and account activation. Track AMC details and all user records.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to represent nature and health.
- Background color: Light beige (#F5F5DC), offering a clean, neutral backdrop.
- Accent color: Warm brown (#A0522D) to highlight important actions and data.
- Body and headline font: 'PT Sans' for a modern and readable interface.
- Use simple and clear icons to represent data categories such as animal health, movement, and finance.
- A clean and intuitive layout for easy navigation and data entry.
- Subtle animations to indicate successful data saves or report generation.
